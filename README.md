# Farm To-Do List App

## Description
This is a full-stack To-Do List application that allows users to manage tasks efficiently. The backend is built using **FastAPI** and **Python**, while the frontend is developed with **React**. Users can create, edit, and delete tasks.

## Technologies Used
- Frontend: React, Vite
- Backend: FastAPI, Python
- Database: MongoDB
- Version Control: Git, GitHub

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Responsive UI

## Installation
## Backend
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload
## Frontend
cd frontend  
npm install  
npm run dev
